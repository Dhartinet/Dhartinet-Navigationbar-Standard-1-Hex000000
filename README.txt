    Dhartinet Navigationbar Standard 1 Hex000000 is a navigation bar for websites and other softwares.  It is extremely easy to incorporate it in your website.
	This navigation bar is completely responsive and can be used on any device.  It can be used for cellphones, tablet PCs, normal computer screens
	and also on bigger screens.  There is no maximum limit of screen size in order to use this software.
	Dhartinet Navigationbar Standard 1 Hex000000 has an amazing feature of dual input.  It means that a separate set of options can be given to big screen users
	and small screen users such as cell phone users, tablet users and many more.  This feature is optional and the developer may keep the same options 
	for all the users.  This greatly increases the versatility of our software.
	
	Dhartinet Navigationbar Standard 1 Hex000000 is designed to give an elite look.  Using it in websites gives a superior look to the websites and greatly makes
	it more appealing.  There is a provision for header to be provided on the navigation bar.  The header may include text and/or images.  For example
	logos of a company or any kind of an organisation can be effectively placed on the navigation bar.
    
    
    Please note: 
    
    The default z-index is 9999.  
    This navigation bar is a fixed type of navigation bar.  
    It is positioned at the top on the screen.
    The user must enter the rest of the content inside the division with id="dhartinet_navigationbar_standard_1_hex000000_e_content"
